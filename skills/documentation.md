## Documentation Types

### 1. README.md
- 프로젝트 개요
- 설치 방법
- 사용법
- 기여 가이드

### 2. API Documentation
- 엔드포인트 목록
- 요청/응답 형식
- 인증 방법
- 예제 코드

### 3. Code Comments
- JSDoc/TSDoc 형식
- 함수 설명
- 파라미터 설명
- 반환값 설명

```typescript
 */
```

### Component Documentation
```typescript
/**
 * [Component Name]
 *

 * 
 * @example
 * <ComponentName prop="value" />
 */
```
